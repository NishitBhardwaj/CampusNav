# CampusNav Assets - QR Codes

This folder contains sample QR codes for testing location initialization.

## QR Code Format:
```
campusnav://building_id/floor_id/x_coordinate/y_coordinate/location_name
```

## Example:
```
campusnav://main_building/floor_0/100/300/Main%20Entrance
```

## Usage:
Print these QR codes and place them at strategic locations for users to scan.
