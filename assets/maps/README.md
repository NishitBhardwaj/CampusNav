# CampusNav Assets - Maps

This folder contains floor plan images and SVGs for indoor navigation.

## Expected Files:
- `main_floor_0.png` - Main building ground floor
- `main_floor_1.png` - Main building first floor
- `lib_floor_0.png` - Library ground floor

## Format Guidelines:
- Use PNG or SVG format
- Recommended resolution: 800x600 pixels minimum
- Include scale reference in pixelsPerMeter
- Ensure walkable areas are clearly defined
