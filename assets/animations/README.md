# CampusNav - Lottie Animations

This folder contains Lottie JSON animation files for the app.

## Required Animations

| File | Purpose |
|------|---------|
| `splash_loading.json` | Splash screen animation |
| `navigation.json` | Navigation compass/arrow |
| `success.json` | Success checkmark |
| `loading.json` | Loading spinner |
| `location_pin.json` | Location marker drop |
| `not_found.json` | Empty state illustration |

## Sources

Get free Lottie animations from:
- [LottieFiles](https://lottiefiles.com/)
- [IconScout](https://iconscout.com/lottie-animations)

## Size Guidelines
- Keep files under 100KB for performance
- Use simple animations (< 5 seconds loop)
- Optimize using LottieFiles editor before adding
